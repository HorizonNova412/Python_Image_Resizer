# Python_Image_Resizer

A simple Tkinter-based application for resizing images to a specified maximum 
dimension, designed for use under Linux.

This project is a practice exercise.

## Features

- Resize images to a maximum width or height while preserving aspect ratio.
- Select input and output directories using a graphical interface.

## Installation

To run Python_Image_Resizer, you need Python and the required dependencies. 
Follow these steps:

1. Clone this repository or download the code.
2. Make sure you have Python installed on your system.
3. Install the required dependencies.

## Requirements

- Python 3.x
- Pillow library
